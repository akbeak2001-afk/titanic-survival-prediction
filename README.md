# Прогнозирование выживания на Титанике

Предсказание выживания пассажиров Титаника с анализом ключевых факторов и сравнением подходов к моделированию.

## Задача

Построить модель, которая по характеристикам пассажира (возраст, пол, класс билета, плата за билет и др.) предсказывает вероятность выживания при крушении корабля. Понимание факторов, влияющих на выживание, помогает выявить закономерности в данных и оценить важность различных признаков.

## Датасет

**Источник:** Titanic - Machine Learning from Disaster (Kaggle)

**Объём:** 
- Тренировочный набор: ~891 пассажир
- Тестовый набор: ~418 пассажиров
- 11 признаков (демография, класс билета, плата, семейные отношения)

**Целевая переменная:** Survived (Yes / No), дисбаланс классов ~38% выживших

**Признаки:**
- `PassengerId` — уникальный идентификатор
- `Pclass` — класс билета (1, 2, 3)
- `Sex` — пол пассажира
- `Age` — возраст
- `SibSp` — количество братьев/сестер и супругов на борту
- `Parch` — количество родителей/детей на борту
- `Ticket` — номер билета
- `Fare` — стоимость билета
- `Cabin` — номер каюты
- `Embarked` — порт отправления

## Подход

**EDA (Exploratory Data Analysis)**
- Анализ пропущенных значений (Age, Cabin, Embarked)
- Исследование распределения признаков
- Корреляционный анализ с целевой переменной
- Анализ влияния каждого признака на выживание

**Feature Engineering**
- Обработка пропущенных значений в Age (медиана по классам)
- Создание бинарных признаков (наличие каюты, имя титула)
- One-Hot Encoding категориальных переменных (Sex, Embarked)
- Удаление неинформативных признаков (PassengerId, Ticket)

**Сравнение моделей**
- Baseline (Dummy Classifier)
- Logistic Regression
- Random Forest
- XGBoost
- Использование 5-fold Stratified Cross-Validation для надёжной оценки

**Тюнинг и оптимизация**
- RandomizedSearchCV по ROC-AUC
- Поиск оптимального порога классификации
- Анализ баланса Precision и Recall

**Интерпретация результатов**
- SHAP для понимания вклада каждого признака
- Анализ важности признаков (feature importance)
- Визуализация предсказаний

## Результаты

| Модель | ROC-AUC (CV) | Accuracy | Precision | Recall |
|--------|-------------|----------|-----------|--------|
| Baseline | 0.50 | 0.62 | — | — |
| Logistic Regression | 0.82 | 0.80 | 0.78 | 0.73 |
| Random Forest | 0.84 | 0.82 | 0.80 | 0.75 |
| XGBoost | 0.86 | 0.84 | 0.82 | 0.78 |

**Ключевые факторы выживания:**
1. **Пол** — женщины имели гораздо выше шансы на выживание
2. **Класс билета** — пассажиры первого класса выживали чаще
3. **Возраст** — дети имели выше вероятность выживания
4. **Стоимость билета** — корреляция с классом билета и выживанием
